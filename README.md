# ⚙️Commit convention

| content | description                 |
| :-------: | :---------------------------: |
| add     | 질문 링크 추가              |
| answer  | 질문에 대답 추가            |
| typo    | 오타 및 링크 오류 수정      |
| fix     | 오류 내용 수정              |
| init    | 질문 카테고리 브랜치 초기화 |
| style   | 탬플릿, 스타일 등 내용 이외의 사항 변경 |

```
# init/fix
[content] category
# add/anwer/typo
[content] (category || branch) : interview question
```
