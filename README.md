# Commit convention

| content | description                 |
| ------- | --------------------------- |
| add     | 질문 링크 추가              |
| answer  | 질문에 대답 추가            |
| typo    | 오타 및 링크 오류 수정      |
| fix     | 오류 내용 수정              |
| init    | 질문 카테고리 브랜치 초기화 |

```
# init/fix
[category] content

[category] content : interview question
```
